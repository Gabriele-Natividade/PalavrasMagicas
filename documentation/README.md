\# Jogo Educativo: Palavras Mágicas



\## Sobre o Projeto



"Palavras Mágicas" é um jogo educativo interativo projetado para auxiliar no processo de alfabetização de crianças. O jogo foca no reconhecimento de objetos, identificação de letras e formação de palavras de uma maneira lúdica e divertida.



\## Objetivo Educacional



\- Desenvolver o reconhecimento visual de objetos.

\- Auxiliar na identificação e associação de letras.

\- Praticar a formação de palavras simples.

\- Estimular o raciocínio lógico e a atenção.



\## Tecnologias Utilizadas



Este projeto foi desenvolvido utilizando apenas tecnologias web puras, conforme os requisitos:



\- \*\*HTML5\*\*: Para a estrutura do conteúdo.

\- \*\*CSS3\*\*: Para a estilização e o design visual.

\- \*\*JavaScript (ES6+)\*\*: Para toda a lógica, interatividade e manipulação do jogo.



Nenhum framework ou biblioteca externa (como React, Vue, jQuery, etc.) foi utilizado.



\## Como Jogar



1\.  Abra o arquivo `componentes/index.html` em qualquer navegador de internet moderno (Google Chrome, Firefox, etc.).

2\.  Na tela inicial, clique em "JOGAR".

3\.  Observe a imagem que aparece e tente formar a palavra correspondente.

4\.  Clique nos balões de letras na parte inferior para preencher os espaços.

5\.  Se a letra estiver correta, ela ficará no lugar. Se estiver errada, o balão piscará em vermelho.

6\.  Complete a palavra para avançar para a próxima fase!



\## Personalização



O jogo foi projetado para ser facilmente personalizável por educadores. Todas as fases, palavras e imagens podem ser editadas através de um único arquivo. Para mais detalhes, consulte o guia `COMO\_PERSONALIZAR.md`.
